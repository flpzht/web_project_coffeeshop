# Coffee Shop - Triple Peaks Library ☕

## Sobre o Projeto

Site para uma cafeteria localizada na biblioteca Triple Peaks. O projeto apresenta informações sobre o estabelecimento, receitas em vídeo, formulário de reserva de mesa e links para redes sociais.

Desenvolvido como parte do bootcamp de Desenvolvimento Web, aplicando HTML, CSS e metodologia BEM.

## Tecnologias Utilizadas

- **HTML5**: Estrutura semântica, formulários com validação
- **CSS3**: Flexbox, posicionamento, transições e efeitos hover
- **Metodologia BEM**: Organização e nomenclatura de classes
- **Google Fonts**: Inter e Noto Serif
- **Normalize.css**: Consistência entre navegadores

## Funcionalidades

✅ Navegação entre seções com âncoras
✅ Vídeos incorporados do YouTube com receitas de café
✅ Formulário de reserva com validação de campos
✅ Checkbox customizado com CSS
✅ Links para redes sociais no footer
✅ Estados hover e active em elementos interativos

## Estrutura de Arquivos
```
project/
├── blocks/          # Arquivos CSS separados por bloco (BEM Flat)
├── images/          # Imagens e ícones
├── pages/           # Arquivo CSS principal (index.css)
├── vendor/          # Bibliotecas externas (normalize.css)
└── index.html       # Página principal
```
